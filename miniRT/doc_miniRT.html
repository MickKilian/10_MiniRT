Names in parenthesis are optional and should be added after the object params. They can be added multiple times and mix with others () args

Capital letter can only be defined once in the file in the mandatory part !

<> = required
<<>> = optionnal

Pattern types are :
  • 0 : Checkboard
  • 1 : Lines longitude
  • 2 : Lines latitude
  • 3 : Spiral 2 colors
  • 4 : Spiral gradient
